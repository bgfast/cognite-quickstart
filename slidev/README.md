# Slidev

**Technical purpose:** Build and export the Atlas AI training slide deck (Markdown → PPTX/PDF) using [Slidev](https://sli.dev/).

## Quick start

```bash
cd slidev
npm install
npm run slides        # dev server + preview
npm run export:pptx   # export to docs/slides/atlas-ai-training-slides.pptx
npm run export:pdf   # export PDF
```

See **[docs/atlas-ai-training-slides-README.md](../docs/atlas-ai-training-slides-README.md)** for the full process and Google Slides export.
